# README

## Docker for Mac
ビルド `docker-compose build`  
起動 `docker-compose up -d`  

## 各 Version
Ruby `ruby 2.6.5p114 (2019-10-01 revision 67812) [x86_64-linux]`  
Rails `Rails 6.0.3.1`  

## データベース
PostgreSQL `psql (PostgreSQL) 11.7 (Debian 11.7-0+deb10u1)`  

## bash への入り方
`docker-compose run web bundle exec bash`
